INFORMATION

Aplicación web de una biblioteca desarrollada con Django REST Framework y React

Esta aplicación permite a los usuarios registrados solicitar préstamos de libros a través de un navegador, donde pueden explorar fácilmente el catálogo disponible en la biblioteca.

Desde el panel de administración, los administradores cuentan con herramientas avanzadas para gestionar tanto los libros disponibles en la base de datos como los préstamos realizados por los usuarios.

  • Django
  • React
