ATOMCRAWLER

Portfolio

INFORMATION

AtomCrawler es una herramienta ETL desarrollada en Python que realiza web scraping en la web oficial de la PRIS Power Reactor Information System para recopilar datos acerca de todos los reactores nucleares alrededor del mundo.

Extrae, procesa y carga los datos recolectados, con los que genera gráficos como por ejemplo: Países con mayor número de reactores nucleares | Capacidad bruta 'MWe', etc.

Además, exporta los datos ya procesados a diferentes formatos de archivo, como CSV o XML.

TECHNOLOGIES

  • Python
  • Pandas
  • Matplotlib
  • BeautifulSoup
