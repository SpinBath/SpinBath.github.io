INFORMATION

Aplicación web para taller mecánico desarrollada con Django, que permite realizar operaciones CRUD básicas.

Esta aplicación facilita a los trabajadores del taller el registro y gestión de clientes, vehículos, servicios ofrecidos y facturación.

De esta manera, tanto el dueño como los empleados pueden tener un mayor control de las operaciones del taller y un seguimiento más eficiente de los vehículos que ingresan.

TECHNOLOGIES

  • Python
  • Django
  • JavaScript
  • SQLite
